# PLAYWRIGHTWITHAI

Collection of Playwright examples, JavaScript fundamentals, and exercises used for learning and demos.

## Contents

- Chapter01_Basics — introductory JavaScript examples
- Chapter02_Java_Concepts — Java-related concept files (JS examples)
- Chapter03_Identifiers_Literals_Operators_Statements — JS operators and statements
- Chapter04_Operators — operator examples
- PLAYWRIGHTWITHAI — nested Playwright project and examples
- Tasks — small practice tasks (FizzBuzz, grade calculator, etc.)

## Requirements

- Node.js 14+ (recommended)
- Git

## Setup

Install dependencies and Playwright browsers:

```powershell
npm install
npx playwright install
```

## Running Playwright tests

```powershell
npx playwright test
```

## Notes

- Screenshots, videos, and traces are ignored via `.gitignore`.
- This repository is primarily educational; adjust scripts as needed for your environment.
