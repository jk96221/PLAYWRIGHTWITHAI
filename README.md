# PLAYWRIGHTWITHAI

Lightweight Playwright examples and learning exercises.

## Contents
- Chapter01_Basics: basic Playwright / JS examples
- Chapter02_Java_Concepts: placeholder

## Requirements
- Node.js (>=16)
- npm or yarn
- Playwright (see install steps)

## Setup
1. Install dependencies:

```bash
npm install
npx playwright install
```

2. Run tests / examples (adjust to your scripts):

```bash
npx playwright test
```

- This repository contains sample scripts in `Chapter01_Basics`.
- Add your tests and CI as needed.
