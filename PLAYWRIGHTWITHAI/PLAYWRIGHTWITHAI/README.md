# PLAYWRIGHTWITHAI

## Overview
This project demonstrates various arithmetic operations in JavaScript. It includes a file that showcases addition, subtraction, multiplication, division, modulus, exponentiation, increment, and decrement operations. The results of these operations are logged to the console.

## Files
- **src/Chapter04_Operators/Arthimatic.js**: Contains JavaScript code for arithmetic operations.
- **src/index.js**: Entry point for the application, used to import and execute functions or modules.
- **.gitignore**: Specifies files and directories to be ignored by Git.
- **package.json**: Configuration file for npm, listing dependencies and project metadata.
- **LICENSE**: Licensing terms for the project.

## Installation
To get started with this project, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/jk96221/PLAYWRIGHTWITHAI.git
cd PLAYWRIGHTWITHAI
npm install
```

## Usage
To run the arithmetic operations, execute the following command:

```bash
node src/Chapter04_Operators/Arthimatic.js
```
