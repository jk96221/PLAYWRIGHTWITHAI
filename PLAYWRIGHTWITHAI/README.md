# PLAYWRIGHTWITHAI

## Overview
This project demonstrates basic arithmetic operations in JavaScript. The primary focus is on performing various calculations and logging the results to the console.

## Files
- **Chapter04_Operators/Arthimatic.js**: Contains JavaScript code for arithmetic operations.
- **package.json**: Configuration file for npm, listing dependencies and scripts.
- **.gitignore**: Specifies files and directories to be ignored by Git.
- **README.md**: Documentation for the project.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/jk96221/PLAYWRIGHTWITHAI.git
   ```
2. Navigate to the project directory:
   ```
   cd PLAYWRIGHTWITHAI
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```

## Usage
To run the arithmetic operations, execute the following command:
```
node Chapter04_Operators/Arthimatic.js
```

This will log the results of the arithmetic operations to the console.
